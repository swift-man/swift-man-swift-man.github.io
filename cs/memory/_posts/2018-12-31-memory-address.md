컴퓨터 과학에서 메모리 주소(memory address)는 메모리 위치에 대한 식별자로, 컴퓨터 프로그램이나 하드웨어 장치가 데이터를 저장하고 나중에 이를 가져오는 장소이다.

## CPU 의 접근
CPU 는 메모리 주소를 저장하고 특정 위치에 접근하기 위해 [<i class="fas fa-link"></i> Register](/cs/cpu/address-register) 를 사용한다.
